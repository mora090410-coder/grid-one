# GridOne Visual Asset Briefs

Status: ready for production
Use with: `gridone-exact-posting-calendar-2026-04-19.md`
Primary channel: Facebook
Secondary use: article hero images, simple ads later, website promo tiles

## Design direction
- Keep GridOne brand colors intact: crimson, gold, cream, dark background
- Make everything readable on mobile first
- Favor clean comparison visuals over busy graphics
- Use plain language in-image, not clever headlines nobody searches for
- Show the product when possible
- Keep each visual useful even if someone sees it without reading the caption
